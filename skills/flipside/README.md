# Flipside Skill

Query blockchain data across 40+ chains, create AI agents for crypto analytics, and build automated data pipelines.

## Installation

```bash
npx skills add FlipsideCrypto/flipside-tools
```

## Requirements

- [Flipside CLI](https://docs.flipsidecrypto.xyz/get-started/cli) installed and authenticated

## What this skill provides

This skill teaches AI agents how to:

- Query blockchain data using Flipside's SQL agents
- Discover tables and schemas across 40+ chains
- Create custom agents for crypto analytics
- Build automated data pipelines
- Work with DeFi, NFT, and token transfer data

## Usage

Once installed, your AI coding assistant will automatically use this skill when working with blockchain data, crypto wallets, DeFi protocols, NFTs, or on-chain analytics.
